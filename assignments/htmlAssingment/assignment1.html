What is HTML?

HTML stands for Hyper Text Markup Language. It is the foundation of web. It's a markup language used to create and structure web pages.

What !DOCTYPE html represnts?

It is a declaration that tells a web browser which version of HTML and document type.

What is head?

It represnts the documents header which stores all the information in Meta tag.

What is meta tag?

It defines metadata of an HTML Document.

What is title tag?

It defines the title of the Document.

What is UTF-8?

UTF-8 stands for (Unicode Transformation Format - 8 bit). It is an encoding system that is used to represent the character in digital format.

What is viewport?

A viewport is the area of web page in which the content is visible to the user. It controls how content is displayed on the web page.

What is body tag?

It contains all the content of the HTML document like images, links, tables, lists, header, footer, hero-section and more.
